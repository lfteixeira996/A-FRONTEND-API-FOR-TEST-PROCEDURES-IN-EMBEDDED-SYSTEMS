class_detaileddescription

Base class for all SW validation test cases.

The base class takes care of the necessary actions for initialising and closing a test, including

  • parsing command line arguments to configure the test
  • starting the NSVF bench
  • loading, starting and initialising the CSW
  • initialising the test environment
  • initialising equipments
  • performing final checks after test execution and freeing resources

The implementation of the initialisation actions is defined by strategies (–> Strategy pattern), that are passed within the InitTestConfig parameter of the init(InitTestConfigE) method.

NOTES ON USAGE: All derived test classes must call the following methods in this order for correct initialisation of the test:

  1. init(InitTestConfigE) or init() for predefined configuration
  2. start()

After execution of all test steps the following method must be called to close the test:

  • end()
